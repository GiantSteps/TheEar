eear

Key-based instrument presented at Sonár MHD 2015.

eear is a smartphone app that analyses incoming audio and provides on-screen real-time musical information about the signal, that can be sent via OSC to a laptop or desktop computer for endless musical manipulation and sound-processing.

Features

At the moment, eear provides tonal information about the music around the user. It can act in continuous mode, providing chord estimations as the music evolves, or it can work in static mode, listening to music during a time period selected by the user, giving an estimate of what could be considered the key of the music.

b) Loudness measure...

c,d) Spectral flatness and centroid, which provide a measure of how pitched or noisy the input signal is.

All these estimators can be expanded to a great number of them. In the future we will enquiry into their practical utility, to select a few very meaninful ones.

All the information above, shown on the smartphone screen, is sent via OSC, so any computer with a DAW or digital music software could use this data to influence the generation of music in real-time fitting the enviromental music. (We will demonstrate this in the presentation).

Future Improvements that we will consider to work on during the Tech fest: implement onsets detection and beat tracking. Even further, implement PTP style clocking server for accurate tempo synchronization scenari.
