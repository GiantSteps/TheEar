eear

Real-Time Assistant for Instant Musical Interaction

View on GitHub Download .zip Download .tar.gz

eear is a smartphone app that analyses incoming audio and provides on-screen real-time musical information about the signal, that can be sent via OSC to a laptop or desktop computer for endless musical manipulation and sound-processing.

Features

For now, eear provides tonal information about the music surrounding the user. It can act in continuous mode, providing chord estimations as the music evolves, or it can work in static mode, listening for a time period selected by the user, delivering a global estimate of what could be considered the musical key of the music.

Furthermore, eear displays basic timbral information, including the degree of noisiness, loudness and spectral centroid (a measure of where pitched content is located) of the sound. This information is sent via OSC so it could easily influence musical developments, lighting applications or image processing.

Extension

On of the great things about eear is that it is a work-in-progress built on very flexible technology. For that reason, the addition of new analytical features is extremely easy, and in the near future eear will allow the user to choose among context-dependent features. Among them, onset and beat detection are in the top of our list.

Technology

eear is built with audio procesing library Essentia, from MUsic Technology Group, Universitat Pompeu Fabra, Barcelona and JUCE, a class library for building rich cross-platform applications and plugins.
